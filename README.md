# 学生教务信息管理系统  微信小程序

## 主要功能

### 👩‍🏫 教师 (Teacher)

- **我的课表**: 查看个人完整授课表，支持按周切换。

### 🎓 学生 (Student)

- **我的课表**: 查看个人课表，支持按周切换。

## 目前只支持微信小程序上查看课程表信息，其他功能尚未开发，待规划
请使用教师/学生账号登录查看课程表
